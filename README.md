Lighthouse Medicals is a full-stack online pharmacy web application developed as a real-world BCA project.  
The platform allows users to browse medicines, upload prescriptions, place orders, and make secure online payments.

## Features
- User registration and authentication
- Medicine browsing and search
- Prescription upload and admin approval
- Cart and order management
- Razorpay payment gateway integration
- Admin dashboard for product and order management
- Delivery status tracking

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Payment Gateway: Razorpay

## Project Type
Industry-oriented academic project (BCA)

## Future Enhancements
- Mobile application (Android & iOS)
- Customer rewards and discounts
- Chatbot for instant customer support
